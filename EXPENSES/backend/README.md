# 🚀 EXP-TRK Backend

Welcome to the **EXP-TRK Backend**, the powerful and scalable API that drives our **Expense Tracking and Financial Management System**. This backend is designed using modern web technologies, ensuring high performance, security, and flexibility. 

## 📌 Project Overview

EXP-TRK is a **full-fledged SaaS expense tracking platform** that allows users to:
- Manage and track expenses
- Set and monitor budgets
- Visualize financial insights with statistics and charts
- Receive email alerts for budget thresholds and important financial activities
- Ensure secure authentication and data privacy

---

## ⚙️ Tech Stack

| Category        | Technology          |
|----------------|--------------------|
| **Backend**    | Node.js, Express.js |
| **Database**   | MongoDB, Mongoose  |
| **Authentication** | JWT, bcrypt.js  |
| **Email Services** | Nodemailer |
| **Task Scheduling** | Node-cron |
| **Validation** | Express Validator |
| **Security**   | Helmet, CORS, Rate-limiting |
| **Logging** | Winston, Morgan |
| **Deployment** | Docker, CI/CD (GitHub Actions) |

## 🚀 Getting Started

### 1️⃣ **Clone the Repository**
```sh
git clone https://github.com/devjasani79/exp-trk-backend.git
cd exp-trk-backend
```

### 2️⃣**Install Dependencies**
```sh
npm install
```

### 3️⃣**Run the Server**
```sh
npm start   # For production
npm run dev # For development (nodemon)
```
